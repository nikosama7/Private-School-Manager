Κατά την εκκίνηση του προγράμματος για πρώτη φορά δημιουργείται μια βάση δεδομένων στον server (localdb)\MSSQLLocalDB.
Ζητούνται username και password για να συνδεθεί κάθε χρήστης.
Οι κωδικοί των χρηστών που δημιουργούνται παραπάνω, υπάρχουν στο αρχείο password.txt.
Για να τερματίσει το πρόγραμμα, πληκτρολογείται exit ως username.
